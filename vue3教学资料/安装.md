# 安装

## 本地安装或使用cdn

下载vue.js后使用script标签引入或直接使用cdn引入

```html
<!-- vue2 -->
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
<!-- vue3 -->
<script src="https://unpkg.com/vue@3"></script>
```

## 使用node.js

```shell
npm i vue
```

## 使用命令行工具创建项目

```shell
npm install -g @vue/cli
```

## 安装vue devtools

首先要安装chrome浏览器

克隆 <https://github.com/vuejs/vue-devtools.git>

运行 npm run build

再在浏览器中引入插件即可
