# 简介安装与使用
## 简介
Element-Ui是一款基于vue作为核心框架的ui（user interface: 就是和用户进行交互并展现一些信息的界面）框架

## 安装
首先通过npm安装到项目中
```
npm install element-ui
```

在主程序中注册如下：
```js
import ElementUI from 'element-ui'; // 引入element-ui
import 'element-ui/lib/theme-chalk/index.css'; // 引入element-ui的样式
import Vue from 'vue'

// 安装element-ui组件
Vue.use(ElementUI)
```

## 使用
使用时当作一般的vue组件使用即可
