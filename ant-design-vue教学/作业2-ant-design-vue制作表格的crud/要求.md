# 要求

表格数据：

```js
let student = {
    id: 1
    name: '',
    sex: '',
    score: 60,
    clazz: ''
}
```

具体要求如下:

- 数据请存储到 vuex 中
- 第一行制作一个工具栏
  - 添加按钮
    - 点击按钮，打开一个表单，表单中录入数据
  - 批量删除按钮
    - 点击按钮，删除以选择的行数据
- 第二行制作一个表格用于显示学生信息
  - 带有checkbox
  - 带有操作列
    - 编辑 可以行内编辑 也可以弹出一个表单进行编辑
    - 删除 单行删除
- 表格请添加分页功能
