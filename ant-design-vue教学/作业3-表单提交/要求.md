# 要求

- 制作一个数据类型如下的表格
  ```js
  let user = {
    name: '姓名',
    sex: '性别',
    birthday: '生日',
    sign: '签名',
    hobbies: '爱好'
  }
  ```
- 使用 `a-form` 表单组件添加数据，并完成表单验证，验证规则如下:
  ```js
  let user = {
    name: ['必填', '长度在2~10字之间', '不能包含"共产党"'],
    sign: ['使用文本域录入数据', '最多3行', '最多15字'],
    hobbies: ['至少2个爱好']
  }
  ```
