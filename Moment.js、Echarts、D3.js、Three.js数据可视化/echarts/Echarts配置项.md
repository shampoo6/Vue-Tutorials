# Echarts配置项
## 如何实现官网demo中的实例
官网实例地址：https://echarts.apache.org/examples/zh/index.html

要使用实例里的内容的话，将实例中的option拷贝到vue项目中的option中即可

## 配置项

官网地址：https://echarts.apache.org/zh/option.html#title

项目名 | 作用 | 例子
--- | --- | ---
title | 图表的标题 | { text: '柱状图动画延迟' }
legend | 图例 | { data: ['bar', 'bar2'] }
